Плавающий чат-виджет «Соня» для сайта: AI-подбор букетов и передача заявок
флористу в amoCRM — тот же движок, что и у Telegram-бота.

== Установка ==

1. Админка WordPress → Плагины → Добавить плагин → Загрузить плагин.
2. Выберите файл sonya-chat.zip → Установить → Активировать.
3. В меню слева появится пункт «Соня». Откройте его и проверьте настройки:
   - Адрес сервера бота: https://144-31-108-55.sslip.io (по умолчанию уже задан).
   - Цвет, заголовок, позиция пузыря — по вкусу.
4. Сохраните. Чат-пузырь появится на всех страницах сайта.

== Как это устроено ==

Плагин — тонкий загрузчик: он вставляет на страницы один тег <script>, который
подгружает виджет с сервера бота. Вся логика (диалог, подбор, заявки в amoCRM)
работает на сервере. Поэтому обновления виджета не требуют переустановки плагина.

== Требования ==

Сервер бота «Соня» должен быть доступен по HTTPS, а домен сайта — добавлен в
список разрешённых источников (WEB_ALLOWED_ORIGINS в .env бота).
